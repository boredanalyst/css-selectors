CSS Selectors

CSS Selectors are tools to specify which elements of your web page will receive a particular styling. This allows you to weave together a stylesheet that only targets the relevant parts of your website.

Hit the right spots with CSS Selectors.

NOTES
lightbulb

If your page has a lot of elements, you might want to style only a few elements at a time. With CSS, you can specify which parts of your web site will receive a particular styling and avoid turning your entire page alarmingly red.

Style all elements with the same tag

NOTES
lightbulb

Sometimes, you just want to apply the same style to all elements, you'll be using type or tag selectors. You use the following syntax for this.

          tag {
          attribute:value;
          }
lightbulb

here's an example of a style that affects the background color of all paragraph elements.

          p {
          background-color:yellow;
          }
lightbulb

Use the (#) symbol to specify a style against elements with a particular ID..

          #id_name {
          attribute:value;
          }
lightbulb

The following syntax affects any element with the mySpan ID.

          #mySpan {
          background-color:yellow;
          }
NOTES

ID selector styles just the elements with a particular ID.

Class Selectors allow you to target elements of a certain class.

lightbulb

The following syntax affects all elements with the yellow class.

          .yellow {
          background-color:yellow;
          }
lightbulb

Use the (*) symbol after the container element to style all elements inside it.

          container * {
          style declarations;
          }
lightbulb

The following syntax affects all elements inside a section tag.

          section * {
          background-color:yellow;
          }
NOTES

Universal selectors style all elements inside a container.
